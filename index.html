Wave Phenomena

My notes from Dr. Hamilton's class. The course is divided into the following topics (also accessible using the buttons above):

  1. Fourier acoustics and angular spectra
    • Linear systems theory
    • 2D spatial Fourier transform
    • Wave propagation from position \((x,y,z=z_0)\) to \((x,y,z=z_0+\Delta z)\)
    • Some helpful tidbits about the Fourier transform
  2. Nearfield acoustical holography
  3. Fraunhofer, Fresnel, and parabolic approximations
  4. Sound beams
  5. Green's functions
  6. Born approximation
  7. Propagation and scattering in moving, periodic, and random media
